CodeAlpha - Car Price Prediction using Machine Learning

Project Overview :

This project predicts car selling prices using Machine Learning
regression models. The workflow includes data preprocessing, exploratory
data analysis (EDA), feature engineering, model training, evaluation,
and prediction.

Objectives :

-   Clean and preprocess car dataset
-   Perform exploratory data analysis
-   Build regression models
-   Compare model performance
-   Predict car prices using trained model

Technologies Used :

-   Python
-   Pandas
-   NumPy
-   Matplotlib
-   Seaborn
-   Scikit-learn
-   Jupyter Notebook

Dataset :

File Used: - car data.csv

Features include: - Car Name - Year - Present Price - Kms Driven - Fuel
Type - Selling Type - Transmission - Owner - Selling Price (Target)

Project Structure :

Project Folder ──> car data.csv ──> Car_Price_Prediction_Project.ipynb
──> requirements_car_price.txt ──> README.txt

Workflow :

1.  Import Required Libraries
2.  Load Dataset
3.  Dataset Overview
4.  Data Preprocessing
5.  Exploratory Data Analysis (EDA)
6.  Feature Engineering
7.  Model Building
8.  Model Evaluation
9.  Prediction
10. Conclusion

Models Used :

-   Linear Regression
-   Decision Tree Regressor
-   Random Forest Regressor

Evaluation Metrics :

-   R² Score
-   MAE
-   MSE
-   RMSE

Key Insights :

-   Present price strongly influences selling price.
-   Car age affects resale value.
-   Fuel type and transmission impact price.
-   Random Forest generally provides strong prediction performance.
-   Feature engineering improves model accuracy.

Conclusion :

Machine Learning can effectively estimate car prices using historical
car attributes and regression techniques.
